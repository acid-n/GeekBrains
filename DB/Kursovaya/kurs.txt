Проект boosty.to

Сервис для взаимодействия с активными фанатами.
Данный сервис предназначен для дополнительной монетизации таких категорий:
- видеоблогеров
- подкастеров
- иллюстраторов
- музыкантов
- журналистов

С помощью чего могут привлекать аудиторию:
- Ранний доступ к контенту
- Влияние на будущий контент
- Эксклюзивный контент
И тп.

Функционал данного сервиса:
- Регистрация
- Создание своей страницы
- Создание уровней подписки
- Создание постов

Как это работает:
Пользователь регистрируется. Создает страницу. Настраивает уровни подписки для своих фанатов. Публикует посты. Привлекает своих фанатов из других социальных сетей. Фанаты покупают месячную подписку для доступа к экслюзивному материалу блогера.

У пользователя есть профиль (user_profile).
У блогера есть страница с подписками (store).

Каждый пользователь может создать свою страницу (store).

Страница блогера также имеет профиль (store_profile).

файл create_tables.sql
